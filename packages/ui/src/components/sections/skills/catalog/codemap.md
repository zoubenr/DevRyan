# packages/ui/src/components/sections/skills/catalog/

## Responsibility
Renders settings subviews for browsing/installing skills from the catalog.

## Design
Catalog UI split into list/detail/action widgets with clear loading/error states.

## Flow
Fetches catalog data, renders searchable entries, and dispatches install/remove actions.

## Integration
Integrated with sections/skills, lib/api, and settings state.
