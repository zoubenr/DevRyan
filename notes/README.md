# Personal Notes

This folder keeps personal notes organized with a small, stable structure.

Decision note: no existing personal notes folder was present in this workspace, so the folder was created at the repository root as `notes/` and kept separate from source code.

## Folders

- `Inbox/` — uncategorized notes to review later.
- `Projects/` — active project-specific notes.
- `Journal/` — dated daily, weekly, or recurring notes.
- `Reference/` — reusable information, checklists, and evergreen notes.
- `Archive/` — old or inactive notes kept for history.

## Naming

- Use dates for time-based notes, such as `2026-05-16-daily-note.md`.
- Use short descriptive names for topic notes, such as `home-network-ideas.md`.
- Prefer lowercase words separated by hyphens.

## Maintenance

- Review `Inbox/` weekly.
- Archive stale notes monthly.
- Add a new top-level folder only when at least 3–5 notes clearly need it.
